ACADEMIC PROJECTS

 ZIGBEE BASED VEHICLE ACCESS 
   CONTROL AND PROTECTING 
                    SYSTEM

B.TECH Mini Project

There are many automobile manufacturing companies in india. Many new models of vehicles are being manufactured. All these manufacturers shows interest of luxury mainly but not on safety. Current driver assistance systems are based on a number of technologies, such as radar, computer vision and sensors. Integrating all of these technologies into a single system is normally a costly and complex solution. I propose a complete Zigbee and RFID based driver assistance system solution that leverages the cost-effective, safety to passengers, low power and secure wireless networking features. By zigbee we can transfer data very fast and reliable. This project gives protection to vehicle driver and passengers in all aspect like vehicle accessing using RFID tag, smoke detection in engine, sudden obstacle detection, detecting alcohol consumption by driver.

IOT BASED HOME AUTOMATION

Diploma Major Project

IOT is one of the fastest developing technology. IOT stands for INTERNET OF THINGS. It is a low cost, efficient and simple way of connecting thinngs using Internet. In this project I have dealed about the automation of home appliances using IOT. I have used a NODE MCU8266 module which is heart of the project. It has ability to connect to the local wifi network available, and loads or appliances of home get connected to circuit using relays. We can use a Blynk app which is open source to everyone. By using this app we can get connected to our home appliances and can control them.

PROJECT LEADER : JEEVAN SAI NIMMALA

PROJECT INCHARGE : S. SRAVANTHI MAM, M.TECH(EPS), Asst.prof. EEE dept
